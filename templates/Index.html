Bem-vindo à VaultMovies API

Documentação da API

Gerenciar Chaves API

Este endpoint permite que você gere, renove ou remova chaves API associadas ao seu IP.

URL: /chaves

Método: POST (Gerar), DELETE (Remover)

Parâmetros: Nenhum

Resposta: Uma chave API gerada ou confirmação de remoção.

Obter Todos os Filmes

Este endpoint retorna uma lista de todos os filmes registrados no banco de dados.

URL: /api/filmes

Método: GET

Parâmetros: Nenhum

Resposta: Uma lista de filmes com suas informações básicas, como título, ano e descrição.

Obter Detalhes de um Filme

Este endpoint permite que você consulte os detalhes completos de um filme específico através do seu ID.

Nota: Para acessar os endpoints de filmes, você precisa de uma chave API válida, que pode ser gerada através do endpoint /chaves.
